# Landing-Page
Landing Page Project for 'The Odin Project' themed around the shop from Deadlock.

Welcome! We do business with everyone, no matter if you are a mole, unicorn, goat, Ixian or even dead!

This is a template landing page website that can be customised for any company or project.

It is structured with html and styled with css. It was a really fun project to practice flexbox and learn the intricacies regarding the layout algorithm.
